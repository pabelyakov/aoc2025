# aoc2025

Solutions for [Advent of Code 2025](https://adventofcode.com/2025) on Python
<br><b>Disclaimer:</b> No quality code. <b>Only fun.</b>
<br>

# quick start

## Prepare

### install dependencies
`poetry install`

### run tests
`poetry run pytest`

### run day
#### run all days
`poetry run python ./src/aoc2025/main.py`

### run day
`poetry run python ./src/aoc2025/days/day01.py`
